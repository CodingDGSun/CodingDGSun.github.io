Java中Collections和Arrays类

Created2021-06-30|Updated2021-06-30
|Post View:

Collections

  • Collections是个java.util下的专用静态类，它包含有各种有关集合操作的静态方法。提供一系列静态方法实现对各种集合的搜索、排序、线程安全化等操作.

Arrays

  • Arrays是个java.util下的专用静态类，它包含有各种有关操作数组 ，提供搜索、排序、复制等静态方法。
