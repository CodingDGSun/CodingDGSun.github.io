Java标识符规则

Created2021-06-28|Updated2021-06-28
|Post View:
  • 由字母、数字、下划线和美元$组成，其中数字不能打头
  • 标识符不能是Java关键字和保留字（const、goto）和特殊的直接量（true、false、null），但可以包含保留关键字和保留字和直接量
  • 不能包含空格
  • 只能包含美元符号$，不能包含@、＃等其他特殊字符
